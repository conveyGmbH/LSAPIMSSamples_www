📋 How to configure Azure AD App Registration?
  1. Create App Registration:
    • Go to Azure Portal → Azure Active Directory → App registrations
    • Click "New registration" or select an existing one
    • Note the Application (client) ID and Directory (tenant) ID
  2. Configure Authentication:
    • Go to "Authentication" section
    • Add a "Single-page application" platform
    • Add redirect URI: https://your-domain.com
    • Enable "Access tokens" and "ID tokens"
  3. Configure API Permissions:
    • Go to "API permissions"
    • Add permission → Dynamics CRM → Delegated permissions
    • Select "user_impersonation"
    • Grant admin consent if required
